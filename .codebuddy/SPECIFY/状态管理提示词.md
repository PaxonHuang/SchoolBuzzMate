SPEC-PINIA-STORE-001
CONTEXT: 设计校园论坛帖子状态管理
RULES:
- 严格遵循Vue3响应式原理
- 禁止在store中直接操作DOM
- 数据必须持久化到本地存储
- 大数据量必须分页加载
TECH STACK: Pinia Vue3 Composition API uni-app-storage
REQUIREMENTS:
1. Store名称: forumStore
2. 状态: posts, currentPost, loading, error, page, pageSize
3. Actions: fetchPosts, fetchPostDetail, createPost, likePost
4. Getters: hotPosts, latestPosts, myPosts
5. 持久化：最近浏览的帖子缓存7天
6. 离线支持：网络断开时显示缓存数据
OUTPUT:
- 完整的store定义文件
- 持久化插件集成代码
- 离线数据处理逻辑
- 类型定义文件（TypeScript）
